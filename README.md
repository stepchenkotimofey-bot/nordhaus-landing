# NordHaus Landing

Демонстрационный лендинг строительной компании для GitHub Pages.

## GitHub Pages

1. Создайте публичный репозиторий на GitHub.
2. Загрузите файлы из корня этого проекта в ветку `main`.
3. Откройте `Settings` -> `Pages`.
4. В `Build and deployment` выберите `Deploy from a branch`.
5. Укажите ветку `main` и папку `/root`.
6. Нажмите `Save`.

После публикации сайт будет доступен по адресу:

`https://USERNAME.github.io/REPOSITORY/`
